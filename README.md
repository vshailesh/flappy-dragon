# Flappy Dragon 
1. This is a learning exercise in Rust. I have used a small gaming library called [bracket_lib](https://docs.rs/bracket-lib/latest/bracket_lib/).
2. This projects works on the back of a `tick` function. Each second the screen refreshes around 60 times, thus refreshing the game state that many times.

## Requirements
1. Your system must have OpenGL Library support. Most modern systems have it.
2. If not, then install it from here [OpenGL Download](https://www.khronos.org/opengl/wiki/Getting_Started#Downloading_OpenGL)
## Steps to run the game
1. Clone the repo
2. open terminal and run `cargo run`.
